# 🚗 Car Price Prediction using Machine Learning

---

## 🏢 Project Overview
This project focuses on building a **car price prediction model** using real-world data from Quikr. It involves **data cleaning, preprocessing, exploratory data analysis (EDA), and machine learning** to predict car prices based on various features.

---

## 📝 Project Description

This project uses **Python** to clean and analyze a car dataset and build a **Linear Regression model** for price prediction. It handles inconsistent data, missing values, and feature engineering to improve model performance.

### 🔍 Key Tasks
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Handling Missing Values & Outliers  
- Model Building (Linear Regression)  
- Model Evaluation (R² Score)  
- Model Deployment using Pickle  

---

## 🛠️ Tools & Technologies

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy  
- **Machine Learning:** Scikit-learn  
- **Techniques:** Linear Regression, OneHotEncoder, Pipeline  
- **Environment:** Jupyter Notebook  

---

## 🎯 Project Goals

- Clean and preprocess raw dataset with inconsistent values  
- Convert categorical data into numerical format  
- Analyze key factors affecting car prices  
- Build a predictive model for car price estimation  
- Achieve good model accuracy and reliability  

---

## 📊 Key Insights

- Car price is highly influenced by **year, company, and fuel type**  
- Data cleaning significantly improved model performance  
- Categorical encoding helped capture relationships between features  
- Outlier removal improved prediction accuracy  

---

## 🤖 Model Performance

- Built using **Linear Regression with Pipeline**  
- Achieved **R² Score up to 0.83**  
- Model trained using optimized random state selection  

---

## 🏁 Conclusion

This project demonstrates how raw, unstructured data can be transformed into a reliable machine learning model.  
Through effective preprocessing and modeling techniques, accurate car price predictions were achieved.
